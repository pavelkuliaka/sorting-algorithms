# Sorting algorithms 📦

A package with various sorting algorithms and tests for them, implemented in Python

A package includes bubble sort, heap sort, quick sort and merge sort algorithms

## Code quality

### Install dev dependencies

`uv sync --extra dev`

### Tests

`uv run pytest`

### Linting

`uv run ruff check .`

`uv run ruff format . --check`

### Typing

`uv run pyright`
